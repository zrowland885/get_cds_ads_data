# get_cds_ads_data

Download script for ECMWF CDS (https://cds.climate.copernicus.eu/ i.e. ERA5) and ADS (https://ads.atmosphere.copernicus.eu/ i.e. CAMS) reanalysis data on pressure and/or surface (not model) levels. Handles time-chunked requests and automated clearing of failed and expired requests. Adapted from https://github.com/LS2D/LS2D/tree/main/ls2d/ecmwf.
